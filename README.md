实现抖音转代码字符舞蹈（前端版）
====

:sunny: 最近，刷抖音的时候，看到了一个用字符跳舞的极乐净土视频，于是出于好奇心，就自己在网上找找参考，然后自己实现了一个。主要原理就是先将每一张图片，也就是每一帧通过灰度转化为灰度图，再将灰度值相近取一个平均灰度，替换成字符，再用定时器进行重复绘画，这样就形成动画。
